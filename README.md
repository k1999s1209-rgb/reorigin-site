## 構成

```

├── index.html          # 全セクション（ナビ/ヒーロー/Mission/Service/強み/会社概要/Contact/フッター）
```

## 実装済みのインタラクション

- ヒーロー：見出しの行送りフェードアップ＋ブルー下線の伸長（load時）。背景は青空グラデ＋太陽の光暈＋山並みSVG
- ナビ：スクロールで白半透明＋ぼかし背景／SPはハンバーガー
- 各セクション：ビューポート進入でフェードアップ（IntersectionObserver）
- 強みカード：stagger（80ms間隔）＋数字カウントアップ
- シグネチャー：ページ左のブルー縦ラインがスクロール進捗で伸長（SPは非表示）
- スマホの改行対策：主要見出し・リード文は `.u-phrase`（inline-block）で文節単位のみ折り返し。PC専用改行は `<br class="pc-only">`
- サービス：タブ（01/02/03）／左右矢印／キーボード←→／スワイプで切替。自動再生なし
- `prefers-reduced-motion: reduce` で全アニメ無効化（即時表示）
